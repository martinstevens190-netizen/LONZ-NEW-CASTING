Quick troubleshooting

Use this build in desktop Chrome first. The diagnostics panel will tell you whether the Cast SDK loaded and whether Chrome sees any Cast devices.

If you are testing an unpublished app on Android TV / Google TV, make sure you registered the TV's Google Cast software serial number, not the generic hardware serial, then wait 15 minutes and reboot the TV.

If Cast state stays at NO_DEVICES_AVAILABLE, the issue is almost always browser support, sender/TV network mismatch, or the wrong device serial being registered.
